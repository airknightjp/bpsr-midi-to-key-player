# BPSR MIDI to KEY Player v1.1.1

## Release Title

BPSR MIDI to KEY Player v1.1.1

## Release Description

v1.1.1では、MIDI解析に基づく和音配置、再生速度に応じた演奏計画、リアルタイム入力を含む連打抑止の改善を追加しました。

### Changes

- `和音配置` を追加しました。約35ms以内の発音を和音として解析し、最高声、低音、共通音、声部進行を考慮して演奏可能な配置へ組み替えます。
- 音域縮小が無効な場合は、低音／通常／高音域を比較し、必要な箇所でのみオクターブ切替を行います。音域縮小が有効な場合はC3-B5内で配置します。
- 現在の再生速度を連続的に評価し、低速時は広い音域を活用し、高速時は頻繁な音域切替を避ける計画へ調整します。再生中の速度変更にも追従します。
- 和音配置の初回解析中に進捗率を表示し、再生中の再計算はバックグラウンドで行うようにしました。連続した設定変更はまとめて処理し、古い計算は中止します。
- `ヒューマナイズ`、`コード・ストラム`、`和音最適化` の表示名を、それぞれ `タイミングの自然化`、`和音内の発音差`、`和音配置` に整理しました。
- `和音内の発音差` と和音配置を併用した場合、最高声、内声、ベロシティを考慮した最大12msの自然な発音差を適用します。元MIDIにある発音差は保持します。
- 再生速度の設定範囲を50-200%から10-200%へ拡張しました。
- 連打抑止を実際の出力間隔で判定するように変更し、再生速度や発音タイミング調整後も50ms未満の同一変換先を抑止します。
- 連打抑止をUSB MIDIキーボードのリアルタイム入力変換とリアルタイム試聴音にも適用しました。
- F11などの単体ショートカットが誤って `Alt+F11` と判定される問題を修正しました。
- 合奏用カウントダウンのゲーム内C3キー押下時間を120msへ延長しました。
- 動作中の `受付開始`／`再生開始` ボタンが、開始後も押下状態を視認できる表示になりました。
- ログタブの名称を `ログ` に変更しました。
- About画面とREADMEのバージョン表記を `1.1.1` に更新しました。
