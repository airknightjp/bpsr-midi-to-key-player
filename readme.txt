BPSR MIDI to KEY Player
=======================

このソフトについて
------------------
BPSR MIDI to KEY Player は、MIDIファイルやUSB MIDIキーボード入力を、Windows上のキーボード入力へ変換するツールです。
MIDI音源再生、MIDIファイルの入力変換、USB MIDIキーボードのリアルタイム入力変換に対応しています。

バージョン
----------
v1.1.0

主な機能
--------
・MIDIフォルダ指定とリロード
・MIDI一覧に曲名、長さ、音域を表示
・MIDI一覧のダブルクリックによるMIDI音源の再生／停止
・MIDIファイルをキーボード入力として再生
・USB MIDIキーボード入力をリアルタイムにキーボード入力へ変換
・テストモード(ログ出力のみ)
・再生位置、MIDI音量、再生速度の調整
・再生速度ラベルのダブルクリックで100%に復帰
・透明度ラベルのダブルクリックで100%に復帰
・トランスポーズ、オクターブシフト
・トランスポーズ／オクターブシフトのラベルをダブルクリックして0に復帰
・音域をC3-B5の3オクターブへ縮める機能
・ヒューマナイズ、コード・ストラム、連打抑止
・トラック／チャンネルを T1 - C1 形式で表示
・トラック／チャンネルごとのオン・オフ
・カウントダウン、カウント音、合奏用のゲーム内キー押下
・カウントダウンとゲーム内キー押下のログ表示
・グローバル開始／終了ショートカット
・ショートカット設定のロック
・C3-B5の3オクターブ分のキーバインド変更
・キーバインドの重複を赤色で表示
・キーバインドのデフォルト復元
・日本語、英語、中国語UI
・スカイブルーを含む複数テーマ
・最前面表示、透明度、ウィンドウ縦幅保存
・[閉じる]でタスクトレイに格納
・単一インスタンス制御
・About画面からバージョン、コピーライト、GitHubリンクを表示

使い方
------
1. BPSR_MIDI_to_KEY_Player.exe を起動します。
2. メニューの「ファイル > MIDIフォルダ指定」からMIDIファイルが入ったフォルダを選択します。
3. MIDI一覧から使用するMIDIを選択します。
4. MIDI音源だけを再生する場合は、MIDI一覧の曲をダブルクリックします。
5. MIDIをキーボード入力として再生する場合は、「MIDI入力変換」の「再生開始」を押します。
6. USB MIDIキーボードをリアルタイム変換する場合は、「リアルタイム入力変換」の「受付開始」を押します。
7. カウントダウン中に、キー入力を送りたいアプリを前面にしてください。
8. 実際のキー入力を送らず確認する場合は、「テストモード(ログ出力のみ)」を有効にしてください。

キーボード入力は、その時点でフォーカスされているアプリに送信されます。

同時利用と連打抑止について
--------------------------
MIDI音源再生とリアルタイム入力変換は同時に使用できます。
MIDIファイルの入力変換とリアルタイム入力変換は同時に使用できません。

連打抑止は、MIDIファイル入力変換とMIDI音源再生に適用されます。
リアルタイム入力変換には適用されません。

テストモードについて
--------------------
テストモードが有効な場合、実際のキー入力は送信されません。変換結果はログ出力のみで確認できます。
リアルタイム入力変換では、受信したMIDIをMIDI音量の設定で発音して確認できます。

キーバインドについて
--------------------
「設定 > キーバインド」から、C3-B5の3オクターブ分の変換先キーを変更できます。
変更はすぐに反映され、設定ファイルへ保存されます。
同じキーが複数の音に割り当てられている場合は赤色で表示されます。

権限について
------------
通常は管理者権限なしで使用できます。
ただし、入力先のアプリが管理者権限で起動している場合、通常権限で起動した本ソフトからはキー入力できない場合があります。
その場合は、入力先アプリと同じ権限で本ソフトを起動してください。

設定ファイルについて
--------------------
音量、再生速度、トランスポーズ、オクターブシフト、カウントダウン、ヒューマナイズ、コード・ストラム、連打抑止、テストモード、テーマ、言語、透明度、ショートカット、キーバインド、ウィンドウ縦幅、最後に読み込んだMIDIフォルダなどの設定は、以下に保存されます。

%APPDATA%\BPSR_MIDI_to_KEY_Player\settings.json

注意事項
--------
・本ソフトはWindows専用です。
・一部のゲームやアプリでは、外部からのキー入力が制限される場合があります。
・MIDIファイルの内容や入力先アプリの仕様によって、期待どおりに入力されない場合があります。
・使用中は、意図しないアプリにキー入力が送信されないよう注意してください。
