# BPSR MIDI to KEY Player v1.1.2

## Release Title

BPSR MIDI to KEY Player v1.1.2

## Release Description

v1.1.2では、表示拡大とセクション表示切替を追加し、拡大時を含むウィンドウサイズの保持を改善しました。アプリアイコンと設定名称も現行仕様へ更新し、リリース前レビューで見つかった重複処理と旧UIコードを整理しています。

### Changes

- 表示メニューに100%、110%、125%、150%、175%、200%の拡大率を追加しました。
- リアルタイム入力変換、MIDI入力変換、詳細設定、プレイヤーの表示／非表示を表示メニューから切り替えられるようにしました。
- 拡大後のウィンドウ幅と高さを保存し、次回起動時に復元するようにしました。
- 演奏補正の表示名称を `タイミングの分散`、`和音の分散`、`和音の再構成` に整理しました。
- アプリアイコンを新しいクジラのデザインへ変更し、旧アイコン資材を削除しました。
- 拡大率と透過率のメニュー操作で処理が二重実行される経路を修正しました。
- 旧Notebook式タブ、旧透過率スライダー、旧ラベル構成向けの到達不能コードと未使用翻訳を削除しました。
- 拡大表示時のMIDI一覧の行高と主要セクションの描画を3言語で確認しました。

### Compatibility

- v1.1.1と同じ `%APPDATA%\BPSR_MIDI_to_KEY_Player\settings.json` をそのまま使用できます。
- MIDI再生、リアルタイム入力、キーバインド、連打抑止、和音の再構成に関する設定キーは変更していません。

### Verification

- unittest: 191件成功
- 構文検査: Python 35ファイル成功
- UI検証: 日本語／英語／中国語 × 100%／125%／200%の9通り成功
- 主要4セクションの表示／非表示切替成功
