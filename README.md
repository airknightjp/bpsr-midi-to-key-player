# BPSR MIDI to KEY Player

[English](README.en.md) | [中文](README.zh-CN.md)

> **多言語対応:** 日本語 / English / 中文 のUI表示に対応しています。

BPSR MIDI to KEY Player は、MIDIファイルやUSB MIDIキーボード入力を、Windows上のキーボード入力へ変換するデスクトップツールです。

BPSR系のキーボード演奏を想定しており、MIDIノートを通常のキーボードキーに割り当て、現在フォーカスされているアプリへ入力します。MIDI音源再生、トラック／チャンネル選択、リアルタイム入力変換、音域調整、キーバインド変更、グローバルショートカット、テーマ変更、タスクトレイ格納にも対応しています。

## 主な機能

- `.mid` / `.midi` ファイルを含むフォルダを読み込み。
- MIDI一覧に曲名、長さ、音域を表示。
- MIDI一覧のダブルクリックでMIDI音源を再生／停止。
- MIDIファイルをキーボード入力として再生。
- USB MIDIキーボード入力をリアルタイムにキーボード入力へ変換。
- テストモードでは実際のキー入力を送らず、ログ出力で確認。
- 音量、再生位置、再生速度(10-200%)を調整。音量／速度ラベルのダブルクリックで100%に復帰。
- 表示メニューからウィンドウの拡大率(100-200%)と透明度を調整。
- トランスポーズ(-12から+12半音)とオクターブシフト(-3から+3)をプレイヤー上部から調整。各ラベルのダブルクリックで0に復帰。
- 音域をC3-B5の3オクターブへ自動的に縮める設定。
- C3-B5範囲外の音は、音域縮小が無効な場合にオクターブ切替キーで処理。
- 演奏補正として、タイミングの分散、和音の分散、和音の再構成を設定。
- 共通設定から連打抑止を設定。
- 使用されているトラック／チャンネルを `TC` ヘッダーと `11` 形式の行で表示。
- トラック／チャンネルの組み合わせごとのオン・オフ。再生中も即時反映。
- サステインペダル CC64 を Space キーへ変換。
- MIDI入力変換の開始前にカウントダウンを設定。
- カウントダウン音、および合奏用のゲーム内キー押下に対応。
- カウントダウンとゲーム内キー押下をログへ表示。
- グローバル開始／中断／終了ショートカットをキー入力で設定。既定値はF9／F10／F11で、ファンクションキー単体にも対応。
- ショートカット設定のロック。
- `設定 > キーバインド` からC3-B5の3オクターブ分のキー割り当てを変更。
- キーバインドの重複を赤色で表示。
- キーバインドをデフォルトへ一括復元。
- 日本語、英語、中国語のUI表示切替。
- スカイブルーを含む複数のカラーテーマ。新規設定時の既定テーマはスカイブルー。
- 最前面表示、拡大後を含むウィンドウサイズの保存、前回MIDIフォルダの自動読み込み。
- 表示メニューから主要4セクションの表示／非表示を切り替え。
- `[閉じる]でタスクトレイに格納` に対応。
- 単一インスタンス制御。
- `その他 > BPSR MIDI to KEY Player について` からバージョン、コピーライト、GitHubリンクを表示。

## メニュー

- `ファイル > MIDIフォルダ指定`: MIDIファイルを含むフォルダを選択します。
- `ファイル > 終了`: アプリを完全終了します。
- `表示`: 拡大率、透明度、最前面表示、各セクションの表示／非表示を変更します。
- `設定`: テーマ、言語、キーバインド、タスクトレイ格納を変更します。
- `その他 > BPSR MIDI to KEY Player について`: バージョン情報とGitHubリンクを表示します。

## 音域について

基本となるBPSRキーボード範囲は C3-B5 です。

- C3-B5 の音はそのまま入力します。
- 低い音域は低音側のオクターブ範囲へ切り替えて入力します。
- 高い音域は高音側のオクターブ範囲へ切り替えて入力します。
- `音域を3オクターブに縮める` を有効にすると、C3-B5範囲外の音をオクターブ単位で移動し、C3-B5内へ収めます。

トランスポーズとオクターブシフトは、MIDIファイル入力変換、リアルタイム入力変換、MIDI音源再生、リアルタイム試聴音に適用されます。ピッチ変更後の音に対して、音域縮小または通常の範囲外処理を行います。

## 和音の再構成

`和音の再構成` を有効にすると、約35ms以内に発音する音を和音として解析し、演奏可能な配置へオクターブ単位で組み替えます。

- 最高声、低音、共通音、声部の並び、前後の声部移動を優先し、声部間の過度な開き、同じ物理キーへの衝突、頻繁な音域切替を抑えます。
- 十分な休符をフレーズ境界として扱い、境界を越えた後は直前の配置に縛られすぎず、新しいフレーズに適した音域を選び直します。
- `音域を3オクターブに縮める` が有効な場合は、C3-B5の範囲内だけで配置します。
- 音域縮小が無効な場合は、低音／通常／高音の各範囲を比較し、必要な箇所だけ `<` または `>` で切り替えます。
- 音域縮小が無効な場合は現在の再生速度を連続的に評価し、低速では切替に使える実時間を活用して広い音域を選び、高速では不自然な連続切替を避けます。再生中の速度変更にも即時追従します。
- 初回の計画作成中は、ステータスに `最適化中 0%` から `最適化中 100%` までの進捗を表示し、完了後に再生時計を開始します。
- 再生中の再計算はバックグラウンドで行い、完成するまで現在の計画で再生を継続します。速度の連続変更は150ms単位でまとめ、古くなった計算は中止します。
- すべての音を同時に配置できない場合は、最高声と低音を優先して一部の重複音または内声を省略します。
- MIDIファイル入力変換とMIDI音源再生に適用されます。リアルタイム入力変換には適用されません。

`和音の分散` も有効な場合、完全に同時刻の和音は最高声を先行させ、内声を外声よりわずかに遅らせます。強い音ほど先行しやすい傾向も加味し、発音差は最大12msです。元MIDIに既に発音差がある音は、そのタイミングを保ちます。`和音の分散` が無効な場合、この発音差は適用しません。

タイミングの分散は和音全体の微小な揺らぎとして別に適用されます。連打抑止は、和音の再構成後に同じ変換先となった音を基準に判定します。

## キーバインド

`設定 > キーバインド` から、C3-B5の3オクターブ分の変換先キーを変更できます。

- キー欄を選択して任意のキーを押すと、そのキーが割り当てられます。
- 同じキーが複数の音に割り当てられている場合は赤色で表示されます。
- `デフォルトに戻す` で現在の既定キーバインドへ一括復元できます。
- 変更はすぐに再生中のMIDIファイル入力変換とリアルタイム入力変換へ反映されます。
- 設定ファイルにはデフォルトから変更された分だけ保存されます。

## 使い方

1. `BPSR_MIDI_to_KEY_Player.exe` を起動します。
2. `ファイル > MIDIフォルダ指定` から、MIDIファイルが入ったフォルダを選択します。
3. MIDI一覧から使用するMIDIを選択します。
4. MIDI音源だけを再生する場合は、MIDI一覧の曲をダブルクリックします。
5. MIDIをキーボード入力として再生する場合は、`MIDI入力変換` の `再生開始` を押します。
6. USB MIDIキーボードをリアルタイム変換する場合は、`リアルタイム入力変換` の `受付開始` を押します。
7. カウントダウン中に、キー入力を送りたいアプリを前面にします。
8. 実際のキー入力を送らず確認する場合は、`テストモード(ログ出力のみ)` を有効にします。

キー入力は、その時点でフォーカスされているアプリに送信されます。

## 同時利用と連打抑止

MIDI音源再生とリアルタイム入力変換は同時に使用できます。
MIDIファイルの入力変換とリアルタイム入力変換は同時に使用できません。

連打抑止は、MIDIファイル入力変換、MIDI音源再生、リアルタイム入力変換に適用されます。MIDIファイルでは再生速度、タイミングの分散、和音の分散の適用後、リアルタイム入力では受信後の実際の出力間隔を基準に、50ms未満となる同一変換先の再発音を抑止します。リアルタイム試聴音にも適用され、抑止した音のnote-offは対応する出力を消さないよう消費します。

## 権限について

通常は管理者権限なしで使用できます。

本ツールは Windows の `SendInput` API を使用してキー入力を送信します。入力先アプリが管理者権限で起動している場合、通常権限の本ツールからの入力がWindows側でブロックされることがあります。その場合は、入力先アプリと同じ権限で本ツールを起動してください。

## 設定ファイル

設定は現在のWindowsユーザーの以下に保存されます。

```text
%APPDATA%\BPSR_MIDI_to_KEY_Player\settings.json
```

設定保存は、書き込み中断による破損を避けるため、原子的な保存処理を行います。

## 動作要件

- Windows
- MIDIファイル再生を使う場合はMIDIファイルを含むフォルダ
- リアルタイム入力変換を使う場合はUSB MIDI入力デバイス
