# BPSR MIDI to KEY Player

[English](README.en.md) | [中文](README.zh-CN.md)

BPSR MIDI to KEY Player は、MIDIファイルやUSB MIDIキーボード入力を、Windows上のキーボード入力に変換するデスクトップツールです。

BPSR系のキーボード演奏を想定しており、MIDIノートを通常のキーボードキーに割り当て、現在フォーカスされているアプリへ入力します。MIDI音源の再生、チャンネル選択、リアルタイムMIDI入力変換、音域調整、グローバルショートカット、テーマ変更、タスクトレイ常駐にも対応しています。

## 主な機能

- `.mid` / `.midi` ファイルを含むフォルダを読み込み。
- 読み込んだMIDIを一覧表示し、曲名と長さを確認。
- MIDI一覧のダブルクリックでMIDI音源を再生・停止。
- MIDIファイルをBPSR形式のキーボード入力へ変換。
- USB MIDIキーボード入力をリアルタイムにキーボード入力へ変換。
- テストモードを有効にしてリアルタイム入力変換を受付開始すると、受信したMIDIをMIDI音量の設定で発音。
- 一時MIDI再生ファイルを作らずにMIDI音源を再生。
- 再生位置シークバーで再生位置を操作。
- MIDI音量スライダーで再生音量を調整。
- 再生速度を50-200%の範囲で調整し、再生中の曲へ即時反映。再生速度ラベルのダブルクリックで100%へ復帰。
- ヒューマナイズで演奏タイミングに小さな揺らぎを加え、再生中でもオン・オフを即時反映。
- コード・ストラムで、同時に鳴る2音以上の和音をランダムな順番の短いずれへ変換。
- 連打抑止で、MIDIファイル変換とMIDI音源再生に含まれる50ms未満の同音連打を無視。
- 読み込んだMIDIで使われているチャンネルだけを表示。
- 再生中でもチャンネルのオン・オフを反映。
- サステインペダル CC64 を Space キーに変換。
- 音域をC3-B5の3オクターブへ自動的に縮める設定。
- トランスポーズ（-12～+12半音）とオクターブシフト（-3～+3）を共通設定から調整し、再生中やリアルタイム入力中にも即時反映。各設定名ラベルのダブルクリックで0へ復帰。
- 音域自動調整を使わない場合は、範囲外の音をオクターブ切替キーで処理。
- MIDI入力変換の開始前にカウントダウンを設定可能。
- カウントダウン中に音を鳴らす設定。
- テストモードでは実際のキー入力を送らず、ログ出力のみで確認。
- 開始・終了ショートカットを、押したキーを検知して設定。
- ショートカットの誤変更を防ぐロック機能。
- 音量、再生速度、トランスポーズ、オクターブシフト、ヒューマナイズ、コード・ストラム、連打抑止、カウントダウン、テストモード、テーマ、ショートカット、ウィンドウ高さ、最後に読み込んだMIDIフォルダなどを保存。
- 次回起動時に前回のMIDIフォルダを自動読込。
- 英語・日本語・中国語のUI表示切替。
- 複数のカラーテーマ。
- ウィンドウ透過度を調整し、透過度ラベルのダブルクリックで100%へ復帰。
- 最前面表示。
- タスクトレイ常駐と、トレイアイコンからの復帰・終了。
- 二重起動を防止し、起動済みの場合は既存ウィンドウを前面表示。

## 音域について

基本となるBPSRキーボード範囲は C3-B5 です。

- C3-B5 の音はそのまま入力します。
- 低い音域は、低音側のオクターブ範囲へ切り替えて入力します。
- 高い音域は、高音側のオクターブ範囲へ切り替えて入力します。
- `音域を3オクターブに縮める` を有効にすると、C3-B5の範囲外の音をオクターブ単位で移動し、C3-B5内に収めます。

この音域調整は、MIDIファイルのキーボード入力変換と、USB MIDIキーボードのリアルタイム入力変換の両方に適用されます。

トランスポーズとオクターブシフトは、MIDIファイル入力変換、リアルタイム入力変換、MIDI音源再生、リアルタイム試聴音のすべてに適用されます。シフト後の音に対して、3オクターブへの音域調整または通常の範囲外処理を行います。

## 使い方

1. `BPSR_MIDI_to_KEY_Player.exe` を起動します。
2. `MIDIフォルダ読込` を押して、MIDIファイルが入ったフォルダを選択します。
3. MIDI一覧から再生したいMIDIを選択します。
4. MIDI音源だけを鳴らしたい場合は、MIDI一覧の曲をダブルクリックします。
5. 選択したMIDIをキーボード入力として再生したい場合は、`MIDI入力変換` の `再生開始` を押します。
6. USB MIDIキーボードをリアルタイム変換したい場合は、`リアルタイム入力変換` の `受付開始` を押します。
7. カウントダウン中に、キー入力を送りたいアプリを前面にします。
8. 実際にキー入力を送らず確認したい場合は、`テストモード（ログ出力のみ）` を有効にします。

キーボード出力は、その時点でフォーカスされているアプリに送信されます。

## 権限について

通常は管理者権限不要で動作します。

キーボード出力には Windows の `SendInput` API を使用しています。通常のデスクトップアプリに対しては、一般ユーザー権限のまま入力できます。ただし、入力先アプリが管理者権限で起動している場合、通常権限の本ツールからの入力がWindows側でブロックされることがあります。その場合は、入力先アプリと同じ権限で本ツールを起動してください。

## 設定ファイル

設定は現在のWindowsユーザーの以下に保存されます。

```text
%APPDATA%\BPSR_MIDI_to_KEY_Player\settings.json
```

設定保存は、書き込み途中で中断されても壊れにくいように、原子的な保存処理を行っています。

## 動作要件

- Windows
- MIDIファイル再生を使う場合はMIDIファイルを含むフォルダ
- リアルタイム入力変換を使う場合はUSB MIDI入力デバイス
