# BPSR MIDI to KEY Player

[日本語](README.md) | [English](README.en.md)

BPSR MIDI to KEY Player 是一款 Windows 桌面工具，可将 MIDI 文件和 USB MIDI 键盘输入转换为键盘输入。

本工具面向 BPSR 风格的键盘演奏。它会把 MIDI 音符映射到普通键盘按键，并发送到当前获得焦点的应用程序。工具还支持 MIDI 声音播放、音轨／通道选择、实时输入转换、音域调整、自定义按键绑定、全局快捷键、主题和托盘收纳。

## 主要功能

- 读取包含 `.mid` / `.midi` 文件的文件夹。
- 在 MIDI 列表中显示名称、时长和音域。
- 双击 MIDI 文件可播放或停止 MIDI 声音。
- 将 MIDI 文件转换为键盘输入。
- 将 USB MIDI 键盘输入实时转换为键盘输入。
- 测试模式下不发送实际按键，只输出日志。
- 调整 MIDI 音量、播放位置和播放速度。双击播放速度标签可恢复为100%。
- 调整窗口透明度。双击透明度标签可恢复为100%。
- 在通用设置中调整移调(-12到+12半音)和八度移位(-3到+3)。双击对应标签可恢复为0。
- 可将音域自动压缩到 C3-B5 的三个八度范围。
- 未启用音域压缩时，C3-B5范围外的音符会通过八度切换键处理。
- 支持演奏人性化、和弦扫弦和防止快速连击。
- 以 `T1 - C1` 格式显示使用中的音轨／通道组合。
- 可单独开关每个音轨／通道组合，并在播放过程中即时生效。
- 将延音踏板 CC64 转换为空格键。
- 可在 MIDI 输入转换开始前设置倒计时。
- 支持倒计时声音，以及合奏用的游戏内 C3 按键输入。
- 将倒计时和游戏内倒计时按键输出到日志。
- 可通过按键输入设置全局开始／结束快捷键。
- 可锁定快捷键设置以防误改。
- 可从 `设置 > 按键绑定` 修改 C3-B5 三个八度的按键分配。
- 重复按键绑定会以红色显示。
- 可一键恢复默认按键绑定。
- 支持日语、英语、中文界面切换。
- 提供包含天蓝色在内的多个颜色主题。新设置默认使用天蓝色。
- 保存置顶、窗口高度和上次读取的 MIDI 文件夹。
- 支持 `[关闭]时最小化到托盘`。
- 防止重复启动。
- 可从 `其他 > 关于 BPSR MIDI to KEY Player` 查看版本、版权和 GitHub 链接。

## 菜单

- `文件 > 指定 MIDI 文件夹`: 选择包含 MIDI 文件的文件夹。
- `文件 > 退出`: 完全退出应用程序。
- `显示`: 修改透明度和置顶显示。
- `设置`: 修改主题、语言、按键绑定和托盘行为。
- `其他 > 关于 BPSR MIDI to KEY Player`: 显示版本信息和 GitHub 链接。

## 音域说明

基础 BPSR 键盘范围为 C3-B5。

- C3-B5 范围内的音符会直接输入。
- 较低音符可切换到低音八度范围输入。
- 较高音符可切换到高音八度范围输入。
- 启用 `缩小到 3 个八度` 后，C3-B5范围外的音符会按八度移动并收进 C3-B5 范围内。

移调和八度移位会应用于 MIDI 文件键盘转换、实时输入转换、MIDI 声音播放和实时试听声音。音高变更后，再执行音域压缩或通常的范围外处理。

## 按键绑定

可通过 `设置 > 按键绑定` 修改 C3-B5 三个八度范围的输出按键。

- 选择按键输入框并按下键盘按键即可分配。
- 重复分配会以红色显示。
- `恢复默认` 会将所有绑定恢复为当前默认映射。
- 修改会立即应用到 MIDI 文件键盘转换和实时输入转换。
- 设置文件中只保存与默认值不同的绑定。

## 使用方法

1. 启动 `BPSR_MIDI_to_KEY_Player.exe`。
2. 通过 `文件 > 指定 MIDI 文件夹` 选择包含 MIDI 文件的文件夹。
3. 从 MIDI 列表中选择要使用的 MIDI。
4. 只播放 MIDI 声音时，双击 MIDI 列表中的曲目。
5. 将 MIDI 作为键盘输入播放时，点击 `MIDI 输入转换` 的 `开始播放`。
6. 实时转换 USB MIDI 键盘时，点击 `实时输入转换` 的 `开始接收`。
7. 倒计时期间，将目标应用程序切换到前台。
8. 只想确认日志而不发送实际按键时，启用 `测试模式(仅输出日志)`。

键盘输入会发送到当时获得焦点的应用程序。

## 同时使用和防止快速连击

MIDI 声音播放和实时输入转换可以同时使用。
MIDI 文件键盘转换和实时输入转换不能同时使用。

防止快速连击适用于 MIDI 文件键盘转换和 MIDI 声音播放，不适用于实时输入转换。

## 权限说明

通常不需要管理员权限。

本工具使用 Windows `SendInput` API 发送键盘输入。如果目标应用程序以管理员权限运行，Windows 可能会阻止普通权限下的本工具发送输入。此时请用与目标应用程序相同的权限级别启动本工具。

## 设置文件

设置保存在当前 Windows 用户的以下位置：

```text
%APPDATA%\BPSR_MIDI_to_KEY_Player\settings.json
```

设置保存采用原子写入方式，以降低写入中断导致设置文件损坏的可能性。

## 运行要求

- Windows
- 使用 MIDI 文件播放时，需要包含 MIDI 文件的文件夹
- 使用实时输入转换时，需要 USB MIDI 输入设备
