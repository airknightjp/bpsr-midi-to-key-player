# BPSR MIDI to KEY Player

[日本語](README.md) | [English](README.en.md)

BPSR MIDI to KEY Player 是一款 Windows 桌面工具，可以将 MIDI 文件和 USB MIDI 键盘输入转换为键盘输入。

它面向 BPSR 风格的键盘演奏：工具会把 MIDI 音符映射为普通键盘按键，并把转换后的输入发送到当前获得焦点的应用程序。它还支持 MIDI 声音播放、通道筛选、实时 MIDI 输入转换、音域调整、全局快捷键、主题切换和托盘常驻。

## 主要功能

- 加载包含 `.mid` / `.midi` 文件的文件夹。
- 在 MIDI 列表中显示已加载的文件名和时长。
- 双击 MIDI 列表中的文件即可播放或停止 MIDI 声音。
- 将 MIDI 文件转换为 BPSR 风格的键盘输入。
- 将 USB MIDI 键盘输入实时转换为键盘输入。
- 启用测试模式并开始实时输入转换后，以 MIDI 音量设置播放接收到的音符。
- 播放 MIDI 声音时不创建临时 MIDI 播放文件。
- 使用播放位置滑块控制播放进度。
- 使用 MIDI 音量滑块调整播放音量。
- 在 50%-200% 范围内调整播放速度，并在播放过程中即时生效。双击播放速度标签可恢复为 100%。
- 通过演奏人性化添加轻微的时间波动，并可在播放过程中即时启用或关闭。
- 对两个或更多同时发音的音符应用和弦扫弦，以随机顺序和短暂时间差播放。
- 启用防止快速连击后，MIDI 文件转换和 MIDI 声音播放会忽略间隔小于 50 毫秒的同音重复。
- 只显示当前 MIDI 文件中实际使用的通道。
- 播放过程中也可以即时应用通道开关。
- 将延音踏板 CC64 转换为空格键。
- 可选择将音域自动缩小到 C3-B5 的三个八度范围。
- 可在通用设置中调整移调（-12 至 +12 个半音）和八度移位（-3 至 +3），并在播放或实时输入过程中即时生效。双击任一设置标签可将其恢复为 0。
- 不启用音域自动调整时，范围外音符会通过八度切换按键处理。
- MIDI 输入转换开始前可设置倒计时。
- 可在倒计时期间播放提示音。
- 测试模式下只输出转换日志，不发送真实键盘输入。
- 通过实际按下按键来设置开始和结束快捷键。
- 可锁定快捷键设置，防止误修改。
- 保存音量、播放速度、移调、八度移位、演奏波动、和弦扫弦、防止快速连击、倒计时、测试模式、主题、快捷键、窗口高度、上次加载的 MIDI 文件夹等设置。
- 下次启动时自动重新加载上次使用的 MIDI 文件夹。
- 支持英语、日语、中文界面。
- 提供多种颜色主题。
- 支持调整窗口透明度，双击透明度标签可恢复为 100%。
- 支持窗口置顶。
- 支持常驻任务托盘，并可从托盘图标恢复或退出。
- 防止重复启动；再次启动时会将已打开的窗口显示到前台。

## 音域说明

基础 BPSR 键盘范围为 C3-B5。

- C3-B5 范围内的音符会直接输入。
- 较低的音符会切换到低音八度范围后输入。
- 较高的音符会切换到高音八度范围后输入。
- 启用 `缩小到 3 个八度` 后，C3-B5 范围外的音符会按八度移动，直到落入 C3-B5 范围内。

该音域调整同时适用于 MIDI 文件键盘转换和 USB MIDI 键盘实时输入转换。

移调和八度移位适用于 MIDI 文件键盘转换、实时输入转换、MIDI 声音播放和实时试听声音。音高移位后，再进行三八度音域调整或常规范围外处理。

## 使用方法

1. 启动 `BPSR_MIDI_to_KEY_Player.exe`。
2. 点击 `加载 MIDI 文件夹`，选择包含 MIDI 文件的文件夹。
3. 在 MIDI 列表中选择要使用的 MIDI 文件。
4. 如果只想播放 MIDI 声音，请双击 MIDI 列表中的文件。
5. 如果想把所选 MIDI 作为键盘输入播放，请在 `MIDI 输入转换` 中点击 `开始播放`。
6. 如果想实时转换 USB MIDI 键盘输入，请在 `实时输入转换` 中点击 `开始接收`。
7. 倒计时期间，将目标应用程序切换到前台。
8. 如果只想确认转换结果而不发送真实按键，请启用 `测试模式（仅输出日志）`。

键盘输出会发送到当前获得焦点的应用程序。

## 权限说明

通常不需要管理员权限。

本工具使用 Windows `SendInput` API 发送键盘输入。对于普通桌面应用程序，使用普通用户权限即可发送输入。但是，如果目标应用程序以管理员权限运行，Windows 可能会阻止来自普通权限进程的输入。此时请以与目标应用程序相同的权限级别启动本工具。

## 设置文件

设置会保存到当前 Windows 用户的以下位置。

```text
%APPDATA%\BPSR_MIDI_to_KEY_Player\settings.json
```

设置保存采用原子写入方式，以降低写入中断导致设置文件损坏的风险。

## 运行要求

- Windows
- 使用 MIDI 文件播放时需要包含 MIDI 文件的文件夹
- 使用实时输入转换时需要 USB MIDI 输入设备
